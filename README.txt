How to setup development environment:
=====================================

First configurure your system for Medienverwaltung Common, see medienverwaltungcommon/README.[yourOS].txt
Then Medienverwaltung Web, see medienverwaltungweb/README.[yourOS].txt


How to configure Android Barcode Scanner app for Medienverwaltung:
==================================================================

- Start Barcode Scanner app
- Press menu
- Look for "Eigene Such-URL" and paste http://hostname:5000/mobile/lookup?q=%s
